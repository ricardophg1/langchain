streamlit==1.29.0

Note: The current environment doesn't have Python with these dependencies installed. To run this application, you'll need to set up a Python environment with the required packages.

How to Run This Application

  1. Ensure Python 3.8+ is installed
  2. Install the required packages:
    pip install -r requirements/prod.txt
  3. Run the Streamlit application:
    cd business-analytics-pro
    streamlit run langchain_project/main.py
